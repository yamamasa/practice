# Introduction
## はじめに
このサイトは私の独断と偏見に基づくベストプラクティスをただただ晒すサイトです。

## 目的
複数人での開発を行うにあたって、スタンダードな考え方や実装方法はたくさんあります。<br>
しかし、細部に至っては、やはり、責任者がどう考えるかということがすごく大事です。<br>
そこで、私とかかわる人たちが、私がどう考えるかを速やかにインプットできるようにするために公開しています。<br>
